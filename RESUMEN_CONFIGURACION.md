### Opción B: Generar APK para instalar sin Expo Go (30 minutos)

#### PASO 1: Guardar el proyecto
- En Emergent, busca **"Save to GitHub"** o **"Export"**
- Guarda el proyecto con nombre: `calculadora-precios`

#### PASO 2: En tu computadora
```bash
# 1. Instala Node.js desde: https://nodejs.org

# 2. Clona el proyecto
git clone [URL-DEL-REPO]
cd calculadora-precios/frontend

# 3. Instala dependencias
npm install

# 4. Instala EAS CLI
npm install -g eas-cli

# 5. Login en Expo (crea cuenta gratis en expo.dev)
eas login

# 6. Genera el APK (tarda 10-15 min)
eas build --platform android --profile preview
```

#### PASO 3: Instala en tu celular
- Descarga el APK del link que te da EAS
- Pásalo a tu celular
- Instala como cualquier APK

## 📋 Perfiles de Build Disponibles

### `preview` (Recomendado para probar)
```bash

```
- ✅ Genera APK directo
- ✅ No necesitas Google Play Console
- ✅ Instalación directa en cualquier Android
- ✅ Ideal para testing

### `production` (Para publicar en tiendas)
```bash
eas build --platform android --profile production
```
- ✅ Genera App Bundle optimizado
- ✅ Para publicar en Google Play Store
- ⚠️ Necesitas cuenta de Play Console ($25)
